# MODEL 1: Global Life Expectancy Predictor

Welcome! This is **MODEL 1** in my Artificial Intelligence portfolio. In this project, I trained a **Machine Learning** algorithm capable of predicting a country's average life expectancy based on its schooling metrics.

**[CLICK HERE TO TRY THE LIVE APP](https://global-life-expectancy-predictor.streamlit.app)**

---

## 📊 Project Features

* **Algorithm Used:** Linear Regression (Continuous Value Prediction).
* **Core Function:** Estimates the exact number of years of life expectancy based on the education level provided.
* **Predictor Variable Used:**
  1. `Schooling` (Average years of schooling).

## 🛠️ Tools & Libraries Used
* **Python** for core development.
* **Pandas** for data exploration and feature selection.
* **Scikit-Learn** for building the linear regression engine and data splitting.
* **Joblib** for exporting and saving the trained model weight.
* **Streamlit Cloud** for hosting and deploying the live interactive web slider app.
